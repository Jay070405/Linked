# 如何打开网站（本地预览）

端口已固定为 **3000**，以后每次打开都是同一个地址。

---

## 方法一：从文件夹双击启动（推荐）

1. 打开 **文件资源管理器**，进入项目里的 `app` 文件夹：
   ```
   C:\Users\shiji\Desktop\curser\portfolio website\app
   ```

2. 在 **地址栏** 输入 `cmd` 然后按回车，会在这个文件夹里打开命令行。

3. 在命令行里输入下面两条命令（可以复制粘贴）：
   ```bash
   npm run dev
   ```

4. 等出现 `✓ Ready in ...` 后，用浏览器打开：
   ```
   http://localhost:3000
   ```

5. 不要关掉这个黑色窗口，关掉网站就停了。要看网站就留着他，不用时关掉即可。

---

## 方法二：从 Cursor / VS Code 打开

1. 在 Cursor 里打开项目后，按 **Ctrl + `**（反引号）打开底部终端。

2. 如果当前不在 `app` 文件夹，先输入：
   ```bash
   cd app
   ```

3. 输入：
   ```bash
   npm run dev
   ```

4. 浏览器打开：**http://localhost:3000**

---

## 方法三：创建桌面快捷方式（可选）

1. 在桌面右键 → 新建 → 快捷方式。

2. “请键入对象的位置”里填（把路径改成你的实际路径）：
   ```
   cmd /k "cd /d C:\Users\shiji\Desktop\curser\portfolio website\app && npm run dev"
   ```
   然后下一步，名字随便写（比如“打开作品集网站”），完成。

3. 以后双击这个快捷方式就会自动开命令行并启动网站；出现 `Ready` 后自己在浏览器打开 **http://localhost:3000**。

---

## 重要说明

- **优先用终端里显示的地址**：运行 `npm run dev` 后，终端里会写 `Local: http://localhost:xxxx`，用这个地址打开浏览器。若 3000 被占用，Next 会用 3010 等，**以终端里显示的为准**。
- **地址固定为**：`http://localhost:3000`（项目里已设成 3000）。若提示“端口 3000 已被占用”，先关掉占用 3000 的程序（或直接使用终端里显示的 3010 等），再重新运行 `npm run dev`。
- 第一次运行如果提示没有 `node_modules`，先在 `app` 文件夹里执行一次：`npm install`，再执行 `npm run dev`。

## 打不开、白屏时

1. **看终端里的地址**：不要只看 3000，若写的是 `http://localhost:3010`，就用 3010 打开。
2. **清缓存再启动**：在 `app` 文件夹里删掉 `.next` 文件夹，然后重新执行 `npm run dev`，再用终端里显示的地址打开。
3. **端口被占用**：想用 3000 的话，在 PowerShell 里执行 `Get-Process -Id (Get-NetTCPConnection -LocalPort 3000).OwningProcess | Stop-Process -Force` 关闭占用 3000 的进程，再 `npm run dev`。
