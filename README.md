# Portfolio Website Workspace

这个仓库现在以 `app/` 目录中的 Next.js 项目为主。

## 开发入口

- 项目目录：`app/`
- 首页入口：`app/src/app/page.tsx`
- 3D 首页：`app/src/components/home/HeroScene.tsx`

## 常用命令

```bash
cd app
npm install
npm run dev
```

浏览器访问 `http://localhost:3000`。

## 目录说明

- `app/src/app/`：页面入口与全局样式
- `app/src/components/`：页面组件
- `app/public/`：图片、视频、简历等静态资源

旧的静态 HTML 版本已不再作为当前开发入口。
